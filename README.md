## Docs?

### Arithmetic operators

```
✿     ADD
❤     SUB
✰     MULT
๑     DIV
{     (
}     )
```

For example: 
12 - (3 + 4) / 7 * 3
would be:
```
12 ❤ {3 ✿ 4} ๑ 7 ✰ 3
```
